Nicotine+ on ArchLinux




Nicotine is in the extra repository, but Nicotine+ is still in AUR

Download the tarball from the Arch User Repository

  • PKGBUILD

Extract and change to the package's directory

tar zxvf nicotine-daelstorm.tar.gz
cd nicotine-daelstorm

Create the package

makepkg

Install the package

pacman -U nicotine-daelstorm-X.X.X-1.pkg.tar.gz
