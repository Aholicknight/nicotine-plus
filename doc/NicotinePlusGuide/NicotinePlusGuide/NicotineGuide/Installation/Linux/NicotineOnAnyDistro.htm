Nicotine+ on Any Distro

  1. Make sure you have all Dependencies (required software) that Nicotine+ needs
  2. Download a Nicotine+ tarball from the Nicotine+ sourceforge page.
  3. Extract the tarball
    tar jxvf nicotine+-X.X.X.tar.bz2
  4. Change to the nicotine directory
    cd nicotine+-X.X.X/
  5. Run nicotine
    ./nicotine
