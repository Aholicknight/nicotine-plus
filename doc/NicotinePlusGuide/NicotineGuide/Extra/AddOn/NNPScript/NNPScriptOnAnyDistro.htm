4.1.1.1 Nicotine+ Extra's: Plug-ins, Now Playing, Any Distro

NNPScript for XMMS

For XMMS you need to install XMMS InfoPipe

XMMS InfoPipe is a plugin that reports XMMS status via named pipe.
Useful if you want to add interesting real-time information for a
personal web page, or a web cam page. more info

XMMS InfoPipe installation

  • 1. download xmms-infopipe source @ official homepage
  • 2. install by: 
    ./configure
     make install
  • 3. restart xmms
  • 4. in xmms: CTRL + P --> General plugins --> enable InfoPipe 1.3 --> OK

Install NNPScript

  • 1. Start Nicotine+
  • 2. In chat input-field:
    /alias np //me 's XMMS |(cat /tmp/xmms-info | grep 'Title\|Status'|cut -d ':' -f2 | tr '\n' ' ' | sed -r 's/^(.+)  (.+)/Now\1: \2/')
  • 3. Test the script by running your XMMS along with Nicotine+. Typ in chat input field the following command.
    /np

NNPScript for MPC

MPC installation

  • 1. You can get MPC via official homepage
  • 2. Install by: 
    ./configure
     make
    make install

Install NNPScript

  • 1. Start Nicotine+
  • 2. In chat input-field:
    /alias op //me 's MPD Now Playing: |(mpc --format "[[%artist% - ]%title%]|file" | head -n 1 | sed -r 's/$/ /')
  • 3. Test the script by running your mpc along with Nicotine+. Typ in chat input field the following command.
    /np

NNPScript for Amarok

NNPScript Installation

  • 1. Start Nicotine+
  • 2. In chat input-field:
    /al np //me is now playing |(artist=`dcop amarok player artist` ; title=`dcop amarok player title` ; echo " $title by $artist")
  • 3. Test the script by running Amarok along with Nicotine+. Typ in chat input field the following command.
    /np

NNPScript for BMP

For BMP you need to install BMP InfoPipe
BMP InfoPipe is a plugin that reports BMP status via named pipe.
Useful if you want to add interesting real-time information for a
personal web page, or a web cam page. more info

BMP InfoPipe installation

  • 1. download xmms-infopipe source @ official homepage
  • 2. download patch against InfoPipe 1.3 that lets you build the plugin for BMP
  • 3. read instructions provided to compile and install infopipe for BMP
  • 3. restart BMP
  • 4. in BMP: CTRL + P --> Plugins General --> enable InfoPipe 1.3 --> OK

Install NNPScript

  • 1. Start Nicotine+
  • 2. In chat input-field:
    /alias np //me 's BMP |(cat /tmp/xmms-info | grep 'Title\|Status'|cut -d ':' -f2 | tr '\n' ' ' | sed -r 's/^(.+)  (.+)/Now\1: \2/')
  • 3. Test the script by running your BMP along with Nicotine+. Typ in chat input field the following command.
