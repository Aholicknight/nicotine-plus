4.1.1.7 Nicotine+ Extra's: Add On, NNPScript, Fedora

NNPScript for Amarok

NNPScript Installation

  • 1. Start Nicotine+
  • 2. In chat input-field:
    /al np //me is now playing |(artist=`dcop amarok player artist` ; title=`dcop amarok player title` ; echo " $title by $artist")
  • 3. Test the script by running Amarok along with Nicotine+. Typ in chat input field the following command.
    /np
