4.1.1 Nicotine+ Extra's: Add On, Now Playing


At the beginning NNPScript was a real script.. but due some issues I dropped it. In fact, it's now easier to get it working! From now on it's a name to indicate command(s) related to Now Playing on Nicotine+.


NNPscript stands for Nicotine Now Playing Script,
which allows you to displaying what you listen to at the moment.

It's a set of commands which extacts info from a mediaplayer
and displays it in Nicotine+

You can keep your favorite mediaplayer as you sometimes
only need to install widely available plug-in





NNPScript is currently available for XMMS, MPC, Amarok and BMP
