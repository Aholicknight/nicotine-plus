Nicotine+ on Suse Linux



Packages

http://linux01.gwdg.de/~pbleser/rpm-navigation.php?cat=Network/nicotine-plus/

Installation

After download run: 
yast -i nicotine+-x.x.x.rpm

Repositories

Nicotine+ for Suse is in suser-guru repositories @ Guru's RPM Site
