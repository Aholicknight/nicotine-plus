2.3 Nicotine+ Installation: Windows


This page is provided 'as is' without warranty of anykind!

To get Nicotine working under windows you need to download and install:

  • Gtk/Win32 (the latest)
  • Nicotine+ windows installer

HowTo install

- download & install gtk+-win32-runtime-2.8.xx-x (xx is current version number, choose the latest)
- download & nicotine+-x.x.x-win32-winXP.exe (xx is current version number, choose the latest)

If you have GAIM installed, the you don't need to install GTK+. Running 2 versions of GTK+ (2.6/8) will make Nicotine+ run VERY SLOW


More info: developers site @ osiris.network
